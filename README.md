# Beveledge - Interior Design Studio Website

A modern, elegant website for Beveledge Interior Design Studio. Showcasing luxurious residential projects and the studio's design philosophy.

## Project Overview

Beveledge is a portfolio website for an interior design studio based in Bengaluru, India. The site features:

- **Home Page**: Hero section with project highlights and studio values
- **Projects Gallery**: Showcase of completed residential projects
- **Individual Project Pages**: Detailed project information with images
- **About Us**: Studio story and founder information
- **Contact**: Contact form and location details

## Featured Projects

1. **Jadav Residence** - 8,200 sq ft villa in Bengaluru
2. **The Home Lounge** - 2,400 sq ft entertainment zone
3. **La Palazzo 209** - 3,250 sq ft luxury apartment
4. **Maison 325** - 3,250 sq ft residential apartment
5. **The Lake Front Residence** - 3,250 sq ft family home
